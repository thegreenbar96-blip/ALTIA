## Descargar

**[⬇️ Descargar ALTIA (Windows)](https://github.com/thegreenbar96-blip/ALTIA/releases/latest)**

1. Descarga `ALTIA_Portable_1.0.0.zip`
2. Descomprime la carpeta
3. Abre `ALTIA.exe`

> Si Windows muestra SmartScreen: **Más información → Ejecutar de todas formas**.

## Características

- **PDF** — unir, dividir, editar páginas, convertir
- **Compresión** — ZIP, 7Z, RAR, TAR…
- **Imágenes** — convertir, redimensionar, comprimir, rotar
- **Video / Audio** — convertir y procesar (requiere FFmpeg)
- **Archivos** — duplicados, renombrado, organización
- **Seguridad** — cifrado, contraseñas, hashes
- **OCR** — extraer texto (requiere Tesseract)
- **Utilidades** — calculadora, conversor, UUID, QR

Todo se procesa **en tu PC**.
